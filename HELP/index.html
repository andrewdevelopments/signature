File Structure

When you will unzip the archive you will find 4 folders:
  • WWW - Theme folder, this you will upload to your server
  • SASS - SASS files, and Grunt Configuration
  • HELP - File you are looking right now
All you need is in WWW, this is the folder you want to upload to your server, and is having the following structure:
  • /fonts
    • FontAwesome Library
    • ET Line Fonts
  • /images
    • Organized folders for images used in Theme
  • /inc
    • PHP File for forms
  • /scripts
    • main.js - Main jQuery File where the magic takes place
    • main.min.js - Minified version of main.js
    • jquery-1.11.2.min.js - jQuery Library, minified version
    • modernizr-2.8.3.min.js - Javascript Library, minified version
    • / components
      • countdown.js
      • counterup.min.js
      • gmap3.js
      • jquery.magnific-popup.min.js
      • masonry.min.js
      • owl.carousel.min.js
      • parallax.min.js
      • SmoothScroll.js
      • waypoints.min.js
  • /styles
    • main.css - Main CSS File, all the styling is in this file
    • main.min.css - Main CSS File minified
    • custom.css - You can use this file to add your own style
    • /colors
  • index.html - Content Slider
  • index-01.html - Image Parallax
  • index-02.html - Video Background
  • paypal-registration.html - PAYPAL Registration Form
  • shortcodes.html - Shortcode Page

Installation

To install this theme, you need a server, copy WWW folder, paste it on your server and done.

Note: After you moved WWW content on your server, you can choose one of the index-* file and rename the file in index.html

Page Section

You can add different classes with page-section class

Options:

parallax - This is a special class used on parallax sections, with this class you must use attr data-background="yourPathToImage/parallaxImage.jpg"

fullscreen - Make the section full height size ( indicated to use only on <div class="hero"> )

overlay - Add background overlay over an image with following classes (dark, light) 
    
    // Simple section
    <section class="section">
      <div class="container">
     
        // Content
    
      </div>
    </section>

    // Section with background
    <section class="section parallax align-center" data-background="images/parallax/parallax-bg1.jpg">
      <div class="container">
     
        // Content
    
      </div>
    </section>

Contact

How to configure contact form:

Note: In config.php we have all the information we need to setup our forms

    

    /* =====================================================
    Subscribe Options
    ===================================================== */

    // Mailchimp API KEY
    $Mailchimp_API_Key = '123456789209318070asiuzi8134809-us1';

    // Mailchimp List ID
    $Mailchimp_List_ID = '09318070asa';

    // Mailchimp Double Opt In
    $Mailchimp_Double_OptIn = false;

    // Mailchimp Send Welcome Message
    $Mailchimp_Welcome = false;

    /* =====================================================
    Contact Form
    ===================================================== */

    // Email address
    $Email_Address = 'youremailaddress@gmail.com';

    // Subject for Contact Form
    $Contact_Form_Subject = 'Contact Form';

How to find your Mailchimp API Key

  1. Login in to your Mailchimp here
  2. Click on your name / username and go to Account
  3. Click on Extras / API Keys
  4. Copy API Key and paste it on $Mailchimp_API_Key

How to find your Mailchimp List ID

  1. Click on Lists, then click on List Dropdonw Status and Settings
  2. On the end of the page you will find your List ID

Google Map

In this section we will learn how to set up google map and some options

Note: You can only have 1 google map per page

Options:

  • data-address - Your location ( 932 Nostrand Avenue, Brooklyn, NY 11225 )
  • data-infowindow - Some information that will go in your infowindow
  • data-coordinates - Your Coordinates for map centering
    
        <div class="map" data-address="932 Nostrand Avenue, Brooklyn, NY 11225" data-infowindow="<strong>932 Nostrand Avenue, Brooklyn, NY 11225</strong> <br>Phone: 0800 981 22319" data-coordinates="40.666056,-73.951035"></div>
        <script src="http://maps.googleapis.com/maps/api/js?sensor=false" type="text/javascript"></script>

FAQ

  1. Is this an Wordpress Theme ?
    No, is just HTML version, and you can install it by copying WWW Content folder on your server
  2. The images are included in download file ?
    No, it's forbidden to include demo images on main download file.
  3. Why contact forms or videos doesn't work ?
    This things works only on a web server
  4. What do I need to worki with Grunt ?
    You need to install grunt in SASS folder and add the following packages: grunt-contrib-cssmin, grunt-contrib-imagemin, grunt-contrib-jshing, grunt-contrib-sass, grunt-contrib-uglify, grunt-contrib-watch
